00033  *  \section Purpose
00034  *
00035  *  Interface for configuration the PIO Parallel Capture peripheral.
00036  *
00037  *  \section Usage
00038  *
00039  *  -# Configurate the interrupt for PIOA, can be done by PIO_InitializeInterrupts()
00040  *  -# Initialize the PIO Parallel Capture API by filing the SpioCaptureInit structur.
00041  *     2 options:
00042  *       - alwaysSampling: for sample data with or without take in account ENABLE pins.
00043  *       - halfSampling: for sample all data or only one time out of two
00044  *  -# Call PIO_CaptureInit() for init and enable the PDC, init the PIO capture.
00045  *  -# Call PIO_CaptureEnable() for enable the PIO Parallel Capture.
00046  *  -# When an interrupt is received, the PIO_CaptureHandler() is call and the respective
00047  *     callback is launch.
00048  *  -# When the transfer is complete, the user need to disable interrupt with
00049  *     PIO_CaptureDisableIt(). Otherway, the PDC will send an interrupt.
00050  *  -# The data receive by the PIO Parallel Capture is inside the buffer passed in the
00051  *     PIO_CaptureInit().
00052  *
00053  */

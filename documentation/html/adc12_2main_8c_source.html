00030 /**
00031  *  \page adc12 ADC12 Example
00032  *
00033  *  \section Purpose
00034  *
00035  *  The adc12 example demonstrates how to use ADC peripheral.
00036  *
00037  *  \section Requirements
00038  *
00039  *  This package can be used with sam3s-ek. To enable full scale measurement
00040  *  of the potentiometer by default configuration. Close jumper jp18 on 1 and 2,
00041  *  the outer pins of it.
00042  *
00043  *  \section Description
00044  *
00045  *  This application is aimed to demonstrate how to use ADC with PDC.
00046  *  The example reserves buffers for transfer. It should start reading
00047  *  converted data first with PDC enabled. The PDC channel will be
00048  *  configured and then    conversion is started.
00049  *
00050  *  It works as a big size buffer for the ADC peripherals, the data
00051  *  will be stored immediately without interfering with the processor.
00052  *  The example will send the converted data to UART to indicate the
00053  *  conversion for several time is done.
00054  *
00055  *  Steps to be implemented:
00056  *  - Initialize ADC with expected parameters.
00057  *  - Configure and enable interrupt for ADC.
00058  *  - Enable PDC reception.
00059  *  - Handle RXBUFF or ENDRX status in ADC interrupt handler.
00060  *
00061  *  \section Usage
00062  *
00063  *  -# Build the program and download it inside the evaluation board. Please
00064  *     refer to the
00065  *     <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6224.pdf">
00066  *     SAM-BA User Guide</a>, the
00067  *     <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00068  *     GNU-Based Software Development</a>
00069  *     application note or to the
00070  *     <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00071  *     IAR EWARM User Guide</a>,
00072  *     depending on your chosen solution.
00073  *  -# On the computer, open and configure a terminal application
00074  *     (e.g. HyperTerminal on Microsoft Windows) with these settings:
00075  *    - 115200 bauds
00076  *    - 8 bits of data
00077  *    - No parity
00078  *    - 1 stop bit
00079  *    - No flow control
00080  *  -# In the terminal window, the
00081  *     following text should appear (values depend on the board and chip used):
00082  *     \code
00083  *      -- ADC12 Example xxx --
00084  *      -- xxxxxx-xx
00085  *      -- Compiled: xxx xx xxxx xx:xx:xx --
00086  *     \endcode
00087  *  -#  The application will output converted value to hyperterminal per second.
00088  *
00089  *  \section References
00090  *  - adc12/main.c
00091  *  - adc.h
00092  */

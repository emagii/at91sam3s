00030 /**
00031  *  \page usart_iso7816 USART ISO7816 Example
00032  *
00033  *  \section Purpose
00034  *  This example sends ISO 7816 commands to a smartcard connected to the
00035  *  evaluation kits on sam3s.
00036  *
00037  *  \section Requirements
00038  *  This example can be used on sam3s evaluation kit. Please connect the
00039  *  smartcard contacts with following pins which could be easily wired from
00040  *  the board.
00041  *  - <b>SAM3S-EK   -- SMARTCARD</b>
00042  *   - PA11       -- RST
00043  *   - TXD1(PA22) -- I/O
00044  *   - SCK1(PA23) -- CLK
00045  *
00046  *  \section Description
00047  *  The iso7816 software provide in this examples is use to transform APDU
00048  *  commands to TPDU commands for the smart card.
00049  *  The iso7816 provide here is for the protocol T=0 only.
00050  *  The send and the receive of a character is made under polling.
00051  *  In the file ISO7816_Init is defined all pins of the card. User must have to
00052  *  change this pins according to his environment.
00053  *  The driver is compliant with CASE 1, 2, 3 of the ISO7816-4 specification.
00054  *
00055  *  \section Usage
00056  *  -# Build the program and download it inside the evaluation board. Please
00057  *     refer to the <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6224.pdf">SAM-BA User Guide</a>,
00058  *     the <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">GNU-Based Software Development</a>
00059  *     application note or to the <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">IAR EWARM User Guide</a>,
00060  *     depending on your chosen solution.
00061  *  -# On the computer, open and configure a terminal application (e.g.
00062  *     HyperTerminal on Microsoft Windows) with these settings:
00063  *        - 115200 bauds
00064  *        - 8 data bits
00065  *        - No parity
00066  *        - 1 stop bit
00067  *        - No flow control
00068  *  -# Connect the card reader to sam3s-ek board:
00069  *        <table border="1" cellpadding="2" cellspacing="0">
00070  *        <tr><td>C1: Vcc:   7816_3V5V </td><td> C5: Gnd</td> <td> C4: RFU</td></tr>
00071  *        <tr><td> C2: Reset: 7816_RST</td> <td>  C6: Vpp</td> <td> C8: RFU</td></tr>
00072  *        <tr><td> C3: Clock: 7816_CLK</td> <td>  C7: 7816_IO</td> </tr>
00073  *        </table>
00074  *     If necessary,another pin must be connected on the card reader for detecting the
00075  *     insertion and removal: 7816_IRQ.
00076  *     On Atmel boards, all this pins can be easily connecting with jumpers.
00077  *  -# Start the application. The following traces shall appear on the terminal:
00078  *     \code
00079  *      -- USART ISO7816 Example xxx --
00080  *      -- xxxxxx-xx
00081  *      -- Compiled: xxx xx xxxx xx:xx:xx --
00082  *      Display the ATR
00083  *     \endcode
00084  *
00085  *   \section References
00086  *  - usart_iso7816/main.c
00087  *  - iso7816_4.c
00088  *  - pio.h
00089  *  - usart.h
00090  *
00091  */

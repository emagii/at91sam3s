SAM3S-EK - Board Description

Purpose

This file is dedicated to describe the SAM3S-EK board.

Contents

  • For SAM3S-EK information, see "SAM3S-EK - Board informations".
  • For operating frequency information, see "SAM3S-EK - Operating frequencies".
  • For using portable PIO definitions, see "SAM3S-EK - PIO definitions".
  • For on-board memories, see "SAM3S-EK - Memories".
  • Several USB definitions are included here, see "SAM3S-EK - USB device".
  • For External components, see "SAM3S-EK - External components".
  • For Individual chip definition, see "SAM3S-EK - Individual chip definition".

To get more software details and the full list of parameters related to the SAM3S-EK board configuration, please have a look at the source file: board.h

Usage

  • The code for booting the board is provided by board_cstartup_xxx.c and board_lowlevel.c.
  • For using board PIOs, board characteristics (clock, etc.) and external components, see board.h.
  • For manipulating memories, see board_memories.h.

This file can be used as a template and modified to fit a custom board, with specific PIOs usage or memory connections.

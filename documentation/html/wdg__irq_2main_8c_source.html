00030 /**
00031  * \page wdg_irq Watchdog with IRQ Interrupt Example
00032  *
00033  * \section Purpose
00034  *
00035  * This example demonstrates user to trigger a watchdog interrupt
00036  * if the software becomes trapped in a deadlock.
00037  *
00038  * \section Requirements
00039  *
00040  * This package can be used with SAM3S evaluation kits.
00041  *
00042  * \section Description
00043  *
00044  * When launched, this program reloads the watchdog at regular intervals
00045  * before the timer underflow occurs, a LED is blinked. User could press
00046  * button1 to make the program run in a infinite loop without
00047  * reloading the watchdog. So a watchdog interrupt will be triggered, and
00048  * "Enter watchdog interrupt." will print to terminal.
00049  *
00050  * \note
00051  * -# User can enable a watchdog reset instead of an interrupt by setting
00052  * WDRSTEN bit in WDT_MR register.
00053  *
00054  * \section Usage
00055  *
00056  * -# Build the program and download it inside the evaluation board. Please
00057  *    refer to the
00058  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6224.pdf">
00059  *    SAM-BA User Guide</a>, the
00060  *    <a href="http://www.atmel.com/dyn/resources/prod_documents/doc6310.pdf">
00061  *    GNU-Based Software Development</a> application note or to the
00062  *    <a href="ftp://ftp.iar.se/WWWfiles/arm/Guides/EWARM_UserGuide.ENU.pdf">
00063  *    IAR EWARM User Guide</a>, depending on your chosen solution.
00064  * -# On the computer, open and configure a terminal application
00065  *    (e.g. HyperTerminal on Microsoft Windows) with these settings:
00066  *   - 115200 bauds
00067  *   - 8 bits of data
00068  *   - No parity
00069  *   - 1 stop bit
00070  *   - No flow control
00071  * -# Start the application.
00072  * -# In the terminal window, the following text should appear:
00073  *    \code
00074  *     -- Watchdog with IRQ Interrupt Example xxx --
00075  *     -- xxxxxx-xx
00076  *     -- Compiled: xxx xx xxxx xx:xx:xx --
00077  *    \endcode
00078  *
00079  * The user could press the button1 to trigger a watchdog interrupt.
00080  *
00081  * \section References
00082  * - wdg_irq/main.c
00083  * - wdt.c
00084  * - wdt.h
00085  */

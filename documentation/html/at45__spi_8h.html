uint32_t AT45_SendCommand ( At45 *  pAt45,
uint8_t  ucCmd,
uint8_t  ucCmdSize,
uint8_t *  pucData,
uint32_t  dwDataSize,
uint32_t  dwAddress,
SpidCallback  pCallback,
void *  pArgument 
)

Sends a command to the dataflash through the SPI. The command is identified by its command code and the number of bytes to transfer (1 + number of address bytes + number of dummy bytes).If data needs to be received, then a data buffer must be provided.

Note:
This function does not block; its optional callback will be invoked when the transfer completes.
Parameters:
pAt45 Pointer to the At45 instance to initialize.
cmd Command code.
cmdSize Size of command code + address bytes + dummy bytes.
pData Data buffer.
dataSize Number of data bytes to send/receive.
address Address at which the command is performed if meaningful.
callback Optional callback to invoke at end of transfer.
pArgument Optional parameter to the callback function.
Returns:
0.

00032  *
00033  *  \section Purpose
00034  *
00035  *  The adc12_temp_sensor example demonstrates how to use the temperature sensor
00036  *  feature inside the microcontroller.
00037  *
00038  *  \section Requirements
00039  *
00040  *  This package can be used with sam3s-ek. To enable full scale measurement
00041  *  ,Close jumper jp18 on 1 and 2,the outer pins of it.
00042  *
00043  *  \section Description
00044  *
00045  *  The adc12_temp_sensor is aimed to demonstrate the temperature sensor feature
00046  *  inside the device. To use this feature, the temperature sensor should be
00047  *  turned on by setting TSON bit in ADC_ACR. The channel 15 is connected to the
00048  *  sensor by default. With PDC support, the Interrupt Handler of ADC is designed
00049  *  to handle RXBUFF interrupt.
00050  *
00051  *  The temperature sensor provides an output voltage (VT) that is proportional
00052  *  to absolute temperature (PTAT). The relationship between measured voltage and
00053  *  actual temperature could be found in Electrical Characteristics part of the
00054  *  datasheet.
